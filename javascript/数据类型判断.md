#### 3. 数据类型判断
- typeof

  > typeof 是一元操作符，放在其单个操作数的前面，操作数可以是任意类型。返回值为表示操作数类型的一个字符串。
  
  -|-
  typeof undefined| undefined
  typeof null | object
  typeof true | boolean
  typeof 1 | number
  typeof 'aa'| string
