#### 3. 数据类型判断
- typeof

  > typeof 是一元操作符，放在其单个操作数的前面，操作数可以是任意类型。返回值为表示操作数类型的一个字符串。
  
  |typeof | 类型 |
  |- |- |
  | typeof undefined | undefined |
  | typeof null | object |
  | typeof true | boolean |
  | typeof 1 | number |
  | typeof 'aa'| string |
  | typeof object | object |
  | typeof function a(){} | function|

- Object.protototype.toString 
  
  > 返回一个由 "[object " 和 class 和 "]" 组成的字符串，而 class 是要判断的对象的内部属性
  
  Object.protototype.toString
  
  |Object.protototype.toString |类型 |
  |- |- |
  |undefined|Undefined|
  |null|Null|
  |'mie'|String|
  |1|Number|
  |true|Boolean|
  |{'a':1}|Object|
  |function a(){}|Function|
  |/a/g|RegExp|
  |[1,2,3]|Array|
  |new Date()|Date|
  |new Error()|Error|
  |"[{'a':1}]"|JSON|
  |Math|Math|
  |arguments|Arguments|
